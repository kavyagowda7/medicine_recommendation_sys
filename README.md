# medicine_recommendation_sys
The Medicine Recommendation System is a machine learning–based application that suggests similar medicines based on their usage and descriptions. It uses Natural Language Processing techniques to convert medicine information into numerical vectors and computes similarity between drugs using cosine similarity. When a user selects a medicine from the Streamlit interface, the system analyzes the similarity matrix and recommends the top five alternative medicines along with their reasons for use and descriptions. This project demonstrates a content-based recommendation approach built using Python, Pandas, Scikit-learn, and Streamlit.
